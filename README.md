# Revision du code de la route

Petite application interactive pour reviser le code de la route a partir des themes du manuel du candidat au permis de conduire.

## Ouvrir l'application

Ouvre `index.html` dans ton navigateur.

## Fonctionnalites

- Banque de 920 questions extraites du PDF
- Quiz par theme ou tous themes confondus
- Correction immediate avec explication
- Suivi du taux de reussite et de la serie de bonnes reponses
- Mode `Mes erreurs` pour retravailler les questions ratees
- Mode `Mini examen` avec 40 questions aleatoires
- Recherche directe par numero de question
- Avertissement sur les questions signalees avec erreur par La Prudence Auto-ecole
- Dessins du PDF affiches avec les questions concernees
- Fiches rapides de revision
- Progression sauvegardee dans le navigateur

## Note sur le PDF

Le PDF numerote les questions jusqu'a 930, mais le texte extrait passe de la question 416 a la question 427. Les questions 417 a 426 ne sont donc pas presentes dans la banque importee.
